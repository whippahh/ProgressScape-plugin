# ProgressScape Plugin

A RuneLite plugin that syncs your OSRS progression data to [ProgressScape](https://whippahh.github.io/ProgressScape/) — an OSRS progression tracker and planner.

## What it does

- Automatically syncs your **quest completion**, **achievement diaries**, and **boss kill counts** on login and logout
- Lets you manually sync your **collection log** with a single button click
- Powers the lookup feature on the ProgressScape site — no manual data entry needed

## How to use

1. Install the plugin from the RuneLite Plugin Hub
2. Log into OSRS — your quests, diaries and boss KCs sync automatically
3. To sync your collection log: open your Collection Log in-game, then click **Sync Collection Log** in the plugin panel
4. Visit [ProgressScape](https://whippahh.github.io/ProgressScape/) and look up your username

## Data collected

This plugin collects and syncs to a remote server:
- Your OSRS username and account type
- Quest completion status
- Achievement Diary completion (Easy/Medium/Hard/Elite)
- Boss kill counts (from in-game chat messages)
- Collection log obtained items (manual sync only)

No bank contents, GE history, or other player data is ever collected.

See the full [Privacy Policy](https://whippahh.github.io/ProgressScape/privacy.html) for details.
